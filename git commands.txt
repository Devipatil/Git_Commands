git init -- It will creates a new git repository.
git add --adds a change in the working directory to the staging area.
git status --displays the state of the working directory and the staging area.
git commit -- what you'll use to take all of the changes that have been made locally and push them up to a remote repository.
git log -- displays committed snapshots.
git branch -- can be used to create a new branch.
git checkout --navigate between the branches created by git branch. 
git config --set Git configuration values on a global or local project level. 
git push -u origin main -- export their local commits and tags to a remote repository.
git clone url --to point to an existing repo and make a clone or copy of that repo at in a new directory, at another location. 
git diff --Comparing changes.
